* 跨平台的客户端,基于Qt/QML开发 
* 后端框架采用Nodejs+Express+mongodb+C++
* 与多多指教社区数据共享

Build
-----

* Qt 5.3.1
* DDui 1.0.0.0
* Node 0.10.x
* mongodb
* express
You can download DDui here: [DDui Builds](http://121.40.201.188:8888/).

* Extract the following and copy to `DDui/`.

```
Debug/
Release/
include/
Resources/
```
```
nodemodules/
model/
routes/
public/
```

* Open `app.js`, then build. (Nodejs0.10.x)
